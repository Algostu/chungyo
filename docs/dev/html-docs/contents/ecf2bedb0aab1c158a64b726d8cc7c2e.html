Diagram

Description

UseCase1

RegisterInfo

  1. 사용자가 본인의 정보를 시스템에 등록한다.

UseCase2

RegisterSkeleton

  1. 등록된 사용자는 자신의 사진이나 웹캠을 통해서 신체정보를 등록할 수 있다.
    • 각도와 인식률이 규정에 맞아야 한다.
  2. 등록된 신체정보는 시스템에 저장된다.

UseCase3

TrainSkeleton

  1. 스켈레톤을 등록한 사용자는 자신이 원하는 운동을 선택한다.
  2. 자신의 몸에 맞춘 운동 자세를 얻는다.

UseCase4

GetRealTimeFeedBack

  1. 사용자는 앱을 실행시키고 자신이 원하는 운동과 운동 속도, 목표갯수를 설정한다.
  2. 옵션을 다 선택했다면, 카메라를 키고 자신을 비춘다.
  3. 화면에서 알려준대로 카메라가 운동을 찍을 방향을 정한다.(고정)
  4. 화면에나온 사람의 자세를 맞추고 3초정도 서있는다.
  5. 화면에서 okay할 경우, 화면에 나온 자세를 보고 따라한다.
    • 화면에서 계속 처음자세에서 넘어가지 않을경우, 계속해서 자세를 잡는다.(수정필요)
  6. 운동을 하다가 틀릴경우 피드백을 전달받는다.
  7. 목표갯수가 다 채워질경우 자동으로 프로그램이 종료된다.
    • 수동으로도 종료가능
  8. 종료후 결과를 확인할 수 있다. 결과는 운동점수, 걸린시간, 목표횟수 달성여부, 등등이 있다.
